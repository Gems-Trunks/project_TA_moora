## Cara install :
1. pastikan ada aplikasi xampp atau laragon
2. taruh file rar di dalam folder htdocs atau www (tergantung pakai aplikasi apa)
3. buka di **Vscode** atau enviorentment lain
4. install php 8.4
5. install composer
6. install npm v22.22.20

## Cara Start Aplikasi :
1. pastikan ada aplikasi xampp atau laragon lalu jalankan
2. jalankan mysql dan apache
3. buka **VScode** masuk ke folder aplikasi nya di www atau htdocs( tergantung aplikasi nya)
4. kemudian jalankan command promnt:
   - php artisan serve
   - npm run dev
   Harus terminal terpisah
5. kemudian masuk ke browser cari:
   - localhost:8000/
   - 127.0.0.1:8000/
   pilih salah satu aja!
6. Aplikasi berjalan dan bisa di pakai
